# Freelance Invoice Generator

Professional invoice generator for freelancers with automated calculations and customizable templates.

## Quick Start

```bash
# Add a client first
node scripts/clients/add.js

# Generate an invoice
node scripts/generate.js

# List all invoices
node scripts/list.js
```

## Features

- Multiple professional templates
- Automated tax calculations
- Client management
- Multi-currency support
- Invoice history tracking

## Money-Making Potential

Save 2-4 hours per week on manual invoice creation, maintain professional branding, and get paid faster.
