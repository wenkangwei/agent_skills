# Weather Skill

A simple weather skill for OpenClaw using wttr.in API.

## Features

- ✅ No API key required
- ✅ Current weather for any location
- ✅ Quick one-liner format
- ✅ Full forecast available

## Installation

This skill is installed in the OpenClaw workspace skills directory.

## Usage

```bash
# Quick check
curl -s "wttr.in/Shanghai?format=3"

# Full forecast
curl -s "wttr.in/Shanghai?T"
```
