# Freelance Proposal Generator

AI 驱动的自由职业提案生成器，帮助快速创建专业提案，提高订单转化率。

## 快速开始

### 1. 生成提案

```bash
node src/generate.js
```

### 2. 查看提案列表

```bash
node src/list.js
```

### 3. 分析成功率

```bash
node src/analyze.js
```

## 功能特性
